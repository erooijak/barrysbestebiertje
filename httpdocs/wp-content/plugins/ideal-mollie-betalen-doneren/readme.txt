Een plugin waarmee u klanten makkelijk donaties kunt laten doen met iDEAL via Mollie.

== Description ==

Met deze plugin kunt u uw klanten makkelijk donaties laten doen
met iDEAL via Mollie. Om deze plugin te kunnen gebruiken heeft
u wel een account bij Mollie nodig. 

U kunt u hier registreren:
http://www.mollie.nl

De plugin werkt uitsluitend met shortcode's die u in posts, pagina's en
widgets kunt plaatsen.

U heeft wel minimaal PHP versie 5.3 nodig om gebruik te kunnen maken
van deze plugin. Er worden functies aangeroepen die niet beschikbaar zijn
in eerdere versies (Zoals goto).

Ook werkt de plugin niet als de http server op een private ip range draait.
Dit is een restrictie van Mollie waar ik niets aan kan doen.
Private IP Ranges vallen in de ranges: 127.x.x.x, 172.x.x.x of 192.x.x.x

== Installation ==
1. Download iDEAL_Mollie_betalen_doneren.zip en pak dit bestand uit.
2. Upload de uitgepakte directory naar uw `/wp-contents/plugins/` directory.
3. Activeer de plugin in het `plugin` menu in WordPress.
4. Na activatie ziet u op uw dashboard een nieuw ikoontje met een euroteken als afbeelding.
5. Daar kunt u verschillende instellingen aanpassen en een uitgebreide helpfile lezen.
